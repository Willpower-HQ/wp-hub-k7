Add an event

Events live in the Notion Event Calendar. Add a row there and it appears here after the next morning sync (about 7am), or ask Claude to run a refresh now.

  1. Open the Event Calendar in Notion
  2. Add a row with the three required fields: Event Name, Date and Event Location (city)
  3. Optional but helpful: Venue, Event Type, Website or Luma link
